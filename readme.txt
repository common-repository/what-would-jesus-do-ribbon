== Description ==


When activated, this plugin will put a What Would Jesus Do Ribbon on the top right corner of your website linking to 12stone.com.
== Installation ==


1. Download the plugin zip archive.

2. Extract it in your `wp-content/plugins` folder or upload via admin panel.

3. Browse to your plugins section and active the plugin.
